Sekretesspolicy

Inledning

Vi "TCT Mobile International Limited" (hädanefter benämnda "vi", "vår", eller "TCT") respekterar din oro avseende sekretess. Denna sekretesspolicy anger våra riktlinjer och rutiner när det gäller personlig och annan information om våra kunder. Vi har för avsikt att ge dig en nivå av komfort och förtroende för hur vi samlar in, använder, avslöjar, och skyddar personlig och annan information som vi samlar in eller som du lämnar till oss. Denna sekretesspolicy förklarar också hur du kan kontakta oss om du har några frågor eller funderingar angående din personliga information.

Omfattning

Denna sekretesspolicy beskriver i detalj vår policy och praxis när det gäller insamling, användning och utlämnande av information som vi samlar in via våra webbplatser, programvara, sociala medier, e-postmeddelanden och ditt inköp eller användning av våra produkter eller tjänster, eller personlig information som du på annat sätt tillhandahåller till oss.

Genom att tillhandahålla personlig information till oss, tillgå vår webbplats eller genom att använda våra produkter eller tjänster, bekräftar du att du har läst och förstått denna sekretesspolicy och accepterat villkoren i denna sekretesspolicy.
Din personliga information lagras och bearbetas på säkra servrar inom Europeiska ekonomiska samarbetsområdet ("EES"). På grund av Internets globala natur kan din personliga information överföras till länder utanför EES vars rättsliga system inte erbjuder samma skyddsnivå för data som lagarna inom den Europeiska unionen och närmare bestämt Frankrike, och genom att använda våra webbplatser, programvara, sociala medier, e-postmeddelanden och ditt inköp eller användning av våra produkter eller tjänster, godkänner du dessa villkor.
Vi vidtar alla åtgärder som rimligen kan krävas för att säkerställa att din personliga information lagras och bearbetas säkert och i enlighet med denna sekretesspolicy.

Den personliga information vi samlar in

Denna sekretesspolicy gäller personlig information, som är information som kan användas för att kontakta eller identifiera en enskild person.
Vi samlar in information du ger oss. Vi tar emot och lagrar säkert all information när du skapar ett konto-ID, köper våra produkter eller tjänster, laddar ner en uppdatering av en programvara, kontaktar oss eller deltar i en online-undersökning.
Vi samlar in automatisk information. Vi tar emot och lagrar vissa typer av information när du interagerar med oss.

Vi samlar in information som används i e-postkommunikation. För att hjälpa oss att göra e-postmeddelanden mer användbara och intressanta, får vi en bekräftelse när e-post från vår e-shoppingwebbplats öppnas, om din dator stöder sådana funktioner. Vi jämför också vår kundlista med listor som erhållits från andra företag, i ett försök att undvika att skicka onödiga meddelanden till våra kunder.

Vi samlar in information från andra källor. Vi kan få information om dig från andra källor, som bemyndigats av dig själv att överlämna sådan information till tredje part och lägga till det till vår kontoinformation.

Hur vi använder din personliga information

  • Den personliga information som vi samlar in gör att vi kan hålla dig uppdaterad gällande våra senaste produktlanseringar, programvaruuppdateringar och kommande evenemang. Om du inte vill finnas med på vår e-postlista, kan du välja bort det när som helst genom att uppdatera dina inställningar.
  • Vi använder också personlig information för att hjälpa oss att skapa, utveckla, driva, leverera och förbättra våra produkter, tjänster, innehåll, reklam och ständigt förbättra din användarupplevelse
  • Vi kan använda din personliga information för att svara på dina frågor och uppfylla dina önskemål.
  • Vi kan använda din personliga information, inklusive födelsedatum, för att verifiera identitet, hjälpa till med identifiering av användare, och för att bestämma lämpliga tjänster. Till exempel kan vi använda födelsedatum för att fastställa åldern på kontoinnehavaren.
  • Emellanåt kan vi använda din personliga information för att skicka viktiga meddelanden, såsom meddelanden om inköp och förändringar i våra regler, villkor och riktlinjer. Eftersom denna information är viktig för din interaktion med oss, kan du inte välja bort att ta emot dessa meddelanden.
  • Vi kan också använda personlig information för interna ändamål såsom granskning, dataanalys och forskning för att förbättra våra produkter, tjänster och kundkommunikation.

Insamling och användning av icke-personlig information

Vi kan även samla in information, som inte på egen hand, medger direkt sammanlänkning med en specifik person. Vi kan samla in, använda, överföra och utlämna icke-personlig information för alla ändamål. Här är några exempel på icke-personliga uppgifter som vi kan samla in:

  • Icke-personliga uppgifter såsom yrke, språk, postnummer, riktnummer, unika enhetsidentifierare, hänvisningsadress, plats och tidszonen där en av våra produkter används så att vi kan bättre förstå kundernas beteende och förbättra våra produkter, tjänster och reklam.
  • Icke-personliga uppgifter om kundaktiviteter på vår hemsida, molntjänster och app butik och från våra andra produkter och tjänster. Denna information sammanställs och används för att hjälpa oss att ge mer användbar information till våra kunder och att förstå vilka delar av vår hemsida, produkter och tjänster som är av störst intresse. Vid tillämpningen av denna policy anses denna typ av data vara icke-personlig information.
  • Information om hur du använder våra tjänster, inklusive sökfrågor, som kan användas för att förbättra relevansen av resultaten från våra tjänster. Utom i begränsade fall för att säkerställa kvaliteten på våra tjänster över Internet, kommer sådan information inte vara associerad med din IP-adress.
  • Med ditt uttryckliga medgivande, kan vi samla in data om hur du använder din enhet och program för att hjälpa applikationsutvecklare att förbättra sina program.

Om vi kombinerar icke-personlig information med personlig information behandlas, den kombinerade informationen som personlig information så länge den förblir kombinerad.

Cookies och andra tekniker

Våra webbplatser, onlinetjänster, interaktiva applikationer, e-postmeddelanden och reklam kan använda "cookies" och andra tekniker. Dessa tekniker hjälper oss att bättre förstå användarnas beteende, ange vilka delar av våra webbplatser människor har besökt, och underlätta och mäta effektiviteten av reklam och webbsökningar. Vi behandlar information insamlad av cookies och andra tekniker som icke- personlig information. Men i den mån IP-adresser eller liknande identifierare anses, enligt lokal lagstiftning, vara personlig information behandlar vi även dessa identifierare som personlig information. På samma sätt, i den mån icke-personlig information kombineras med personlig information behandlar vi den kombinerade informationen som personlig information inom ramen för denna sekretesspolicy.
TCT och våra partners använder också cookies och andra tekniker för att komma ihåg personlig information när du använder vår hemsida, onlinetjänster och applikationer. Vårt mål i dessa fall är att göra din upplevelse med TCT mer bekväm och personlig. Till exempel, att känna till att någon som använder datorn eller enheten har handlat en viss produkt eller använt en viss tjänst hjälper oss att göra vår annonsering och e-postkommunikation mer relevant för dina intressen.
Du kan ställa in din webbläsare att acceptera alla cookies, avvisa alla cookies, eller meddela dig när en cookie skickas. Alla webbläsare är olika, så kontrollera "Hjälp"-menyn i din webbläsare för att lära dig hur man ändrar inställningarna för cookies. Till exempel kan du på Microsoft Internet Explorer stänga av eller ta bort cookies genom att välja "Verktyg / Internet-alternativ".
I vissa av våra e-postmeddelanden använder vi en "klickbar URL" kopplat till innehåll på vår hemsida. När kunderna klickar på en av dessa webbadresser, passerar de genom en separat webbserver före ankomsten till destinationssidan på vår webbplats. Vi spårar dessa klickdata för att hjälpa oss att avgöra intresset för särskilda ämnen och mäta effektiviteten i vår kundkommunikation. Om du föredrar att inte spåras på detta sätt, ska du inte klicka på text eller grafiska länkar i e-postmeddelanden.

Delar vi informationen?

Vi delar inte personlig information med företag, organisationer eller privatpersoner utanför TCT såvida inte någon av följande omständigheter gäller.
TCT kan dela personlig information med ditt samtycke.

Vi kan lämna ut personlig information till våra dotterbolag vid behov för att utföra tjänster för vår eller för din räkning.

Vi samarbetar med företag och privatpersoner för att utföra funktioner för vår räkning. Exempel omfattar verkställande av beställningar, leverans av paket, skicka post och e-post, ta bort repetitiva uppgifter från kundlistor, analysera data, erbjuda hjälp med marknadsföring, ge sökresultat och länkar (inklusive betalda listor och länkar), bearbetning av kreditkortsbetalningar, samt ge kundservice. De har tillgång till personlig information som behövs för att utföra uppgifter, men får inte använda den för andra ändamål.

Vi delar personlig information med organisationer eller personer utanför organisationen om vi tror att åtkomst, användning, bevarande eller utlämnande av informationen är nödvändigt för att:

  • uppfylla alla tillämpliga lagar, juridiska processer eller tvingande begäranden av myndigheter.
  • upptäcka, förebygga eller på annat sätt hantera bedrägerier, säkerhetsproblem eller tekniska problem.
  • skydda mot skador på rättigheter, egendom eller säkerhet för våra användare eller allmänheten som krävs eller tillåts enligt lag.

Din personliga information kan överföras till, och lagras vid, en destination utanför EES inom TCT-koncernen. Den kan också bearbetas av personal som arbetar utanför EES som arbetar för oss eller för en av våra leverantörer. Vi vidtar alla åtgärder som rimligen kan krävas för att säkerställa att din personliga information behandlas säkert och i enlighet med denna integritetspolicy.

Säkerhet och transparens

Vi strävar efter att vidta alla rimliga fysiska, tekniska och organisatoriska åtgärder och förfaranden för att skydda och säkra den personliga information som vi samlar in genom våra produkter och vår webbplats.

Vi arbetar hårt för att skydda vårt företag, rykte och våra användare från obehörig åtkomst till eller obehörig ändring, utlämning eller förstörelse av den personliga information vi har. I synnerhet:

  • Vi granskar vår informationsinsamlings-, lagrings- och bearbetningsmetoder, inklusive fysiska säkerhetsåtgärder för att skydda mot obehörig åtkomst till system;
  • Vi använder kryptering när det är lämpligt;
  • Vi använder lösenordsskydd när det är lämpligt; och
  • Vi begränsar åtkomsten till personlig information till våra anställda, leverantörer och agenter som behöver känna till informationen för att kunna behandla den för oss och som är underkastas stränga avtalssekretesskrav. De kan drabbas av disciplinåtgärder eller deras avtal kan sägas upp om de inte uppfyller dessa krav.
    Men dessa åtgärder medför inte att du inte vidtar lämpliga åtgärder för att hålla din personliga information säker och trygg, särskilt genom att undvika uppenbara användar-ID och lösenord, genom att se till att du inte avslöja ditt/dina lösenord eller beviljar någon annan person tillgång till ditt användarkonto och din personliga information, och andra skyddsåtgärder som är tillämpliga.

När det gäller transparens är vårt mål att vara tydliga med vilken information vi samlar in, så att du kan avgöra hur den används.

Om du delar din personliga information via en av våra webbplatser eller enheter med ett tredjepartskonto, som Facebook, Google + eller Twitter, observera att sekretessen för din personliga information kommer att hanteras av den respektive tredje parten. Vi är inte ansvariga för eventuella skador på grund av förlust av data, skada eller intrång som kan uppstå på dina tredjeparts-profiler över vilka vi inte har någon kontroll.

All personlig information eller innehåll som du frivilligt avslöjar när du använder vår webbplats eller enheter och som kan ses av medlemmar i tredjepartsapplikationer eller webbplatser (som chattfunktioner, i meddelanden), blir offentlig och kan hämtas, visas och användas av andra. Du är ansvarig för den personliga informationen som du väljer att dela eller skicka i dessa fall.

Barn

Vi förstår vikten av att vidta extra försiktighetsåtgärder för att skydda integriteten och säkerheten för barn som använder våra produkter och tjänster. Därför samlar vi inte medvetet in, använder eller lämnar ut personlig information från barn under 13 år eller motsvarande lägsta ålder i relevant jurisdiktion, utan verifierbart godkännande från förälder. Om vi får reda på att har vi samlat in personlig information om ett barn under 13 år, eller motsvarande minimiålder beroende på jurisdiktion, utan att först ta emot verifierbart samtycke från förälder vidtar vi åtgärder för att radera informationen så snart som möjligt.

Ändringar

Vår sekretesspolicy kan emellanåt ändras. Vi publicerar eventuella ändringar i sekretesspolicyn på den här sidan och om ändringarna är betydande, erbjuder vi ett mer framträdande meddelande (inklusive, för vissa tjänster, e-postmeddelanden av skretesspolicyförändringar). Vi kommer också att behålla tidigare versioner av denna sekretesspolicy i ett arkiv för din referens.

Den nuvarande versionen är daterad den 11 april 2016.

Lagring av personlig information

Vi håller din personliga information under en begränsad tid som behövs för att uppfylla de syften som beskrivs i denna sekretesspolicy om inte en längre period krävs eller tillåts enligt lag och då endast för de ändamål som angivits ovan. När syftet väl är uppnått, raderas all personlig information på ett säkert sätt.

Platsbaserade tjänster

För att tillhandahålla platsbaserade tjänster för TCT-produkter kan TCT och våra partners och licenstagare samla in, använda och dela exakta lokaliseringsuppgifter, inklusive den geografiska platsen för din enhet i realtid. Ifall tillgängligt, kan platsbaserade tjänster använda GPS, Bluetooth och din IP-adress, tillsammans med publika Wi-Fi-hotspots och mobilmastlokaliseringar och andra tekniker för att avgöra dina enheters ungefärliga position. Om du inte ger ditt samtycke, är denna platsdata som samlas in anonym i en form som inte identifierar dig och används av TCT och våra partners och licenstagare för att tillhandahålla och förbättra platsbaserade produkter och tjänster. Till exempel kan enheten dela sitt geografiska läge med applikationsleverantörer när du väljer att använda deras lokaliseringstjänster.

Tredje parts webbplatser och tjänster

TCT:s webbplatser, produkter, applikationer och tjänster kan innehålla länkar till tredjeparts webbplatser, produkter och tjänster. Våra produkter och tjänster kan också använda eller erbjuda produkter eller tjänster från tredjeparter. Information som samlas in av tredjeparter, som kan innefatta sådant som platsdata eller kontaktuppgifter, styrs av deras integritetspolicy. Vi uppmuntrar dig att informera dig om tredje parts sekretesspolicy.

Dina rättigheter: Åtkomst, redigera, blockera

Du kan få åtkomst, granska och redigera din personliga information när som helst genom att logga in på ditt användarkonto, besöka din användarprofil eller genom att kontakta oss direkt. Om du begär att vi ska redigera eller ta bort din personliga information kan vi be dig att tillhandahålla bevis för din identitet innan vi kan behandla din begäran.

Du kan också få dina data raderade eller blockerade, i förekommande fall, om du anser att dina data är felaktiga.

För alla eller eventuella förfrågningar som definierats ovan kan du skicka din förfråga till följande e-postadress: privacy@alcatelonetouch.com

Under vissa jurisdiktioner och dataskyddslagar, kan du ha rätt att begära information om din personliga information som vi håller, om dig eller att begära rättelse eller radering av din personliga information.

Hjälp oss att se till att den information vi har om dig är korrekt och aktuell. Om du anser att något vi har lagrat om dig är felaktigt eller ofullständigt, vänligen kontakta oss på privacy@alcatelonetouch.com eller skriv till vår kundtjänst.

Frågor om integritet

Om du har frågor eller funderingar kring vår sekretesspolicy eller databearbetning eller om du vill anföra klagomål om ett möjligt brott mot lokala integritetslagar, var vänlig kontakta oss. Du kan alltid kontakta oss via telefon genom det aktuella telefonnumret eller e-postadressen för ditt land.

All kommunikation undersöks och svar utfärdas i förekommande fall så snart som möjligt. Om du är missnöjd med svaret som getts kan du hänvisa ditt klagomål till den relevanta regulatorn i din jurisdiktion. Om du frågar oss, strävar vi efter att förse dig med information om relevanta klagomålsvägar, som kan vara tillämpliga för dina omständigheter.
