Támogatott telefonok

  1. A jövõben az óra további telefonokat is támogatni fog.
  2. Az óra Bluetooth 4.0 LE képes telefonokkal párosítható.
  3. Az óra az Android 4.3 vagy újabb, illetve az iOS7 vagy újabb operációs rendszereket használó telefonokkal kompatibilis.
