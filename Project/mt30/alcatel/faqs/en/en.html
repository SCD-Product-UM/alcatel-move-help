FAQs for MOVETIME Family Watch

Unable to pair with your watch

  1. Make sure the SIM is inserted correctly.
    • If the SIM is inserted correctly, check to see if G, E, or 3G is displayed on the top left of the screen of your watch.
    • If not, try restarting your watch.
  2. If the issue remains unresolved, contact your network operator or configure APN settings.
  3. If G, E, or 3G is displayed, check to make sure there are no overdue network charges that need to paid.

Unable to obtain the position of your watch

  1. Check to see if G, E, or 3G is displayed on the top left of the screen of your watch.
  2. If not, try restarting your watch.
  3. If the issue remains unresolved, contact your network operator or configure APN settings.
  4. If G, E, or 3G is displayed, check to make sure there are no overdue network charges that need to paid.

Unable to obtain the verification code

You may need to activate the relevant SMS receiving service. For more information, contact your network operator.

Waterproof notice

This product has an IP67 rating which means it is fully protected from dust and can withstand being submerged in 1 meter of static water for up to 30 minutes. DO NOT use it when swimming, diving, scuba diving and taking shower.

Positioning accuracy

Your watch will provide four different positioning tools: GPS, AGPS, WLAN, LBS. Position accuracy will differ according to different positioning method.

Data service

The watch uses mobile data to send messages and location information to family members. The data charges incurred will need to be paid by you to the network operator.

SOS

All family members and the child should be familiar with the SOS operation. It is recommended to practice using the operation first.

Whether the material of this product is harmful to children

All materials used have passed the required safety tests.

Charging

Keep the charging port clean. If the charging port is corroded or dirty the charging process will be seriously affected.

Storage and use of location data for user

For all data related to you, we will only use the queries you submit to us. We will not use your data for any commercial application or development without authorisation.

How to add family members into the contacts list?

There are two ways to add family members.

  • Adding a family member manually
    1. Touch Me on the application main screen.
    2. Select a Family watch.
    3. Touch Family watch contacts > Family members.
    4. Touch and then select Add manually.
    5. Enter a name/title and number. You can also upload a photo (optional).
    6. Touch Save.
  • Adding a family member via QR code
    1. Touch Me on the application main screen.
    2. Select a Family watch.
    3. Touch Family watch contacts > Family members.
    4. Touch and then select Scan QR code.
    5. A QR code will appear on screen. If the invited family member is next to you they can scan it directly using their TCLMOVE app. If the invited family member is not in your vicinity, touch to send them the QR code.

How to add and delete safe zones?

  1. Touch Me.
  2. Select a Family watch and touch Safe zone.
  3. Touch at the right top of this page, enter a name, select an location on the map and set the range.
  4. Touch Save.
  5. Select a safe zone and swipe left to delete it.

How to add and delete Alarms?

  1. Touch Me.
  2. Select a Family watch and touch Reminders.
  3. Touch Alarms > > Set the alarm time > Set which days the alarm will repeat.
  4. Touch Save.
  5. Select an alarm and swipe left to delete it.

How to add and delete Reminders?

  1. Touch Me.
  2. Select a Family watch and touch Reminders.
  3. Touch Reminders > > Set the alarm time > Set which days the alarm will repeat.
  4. Touch Save.
  5. Select a reminder and swipe left to delete it.

How to set a school time?

  1. Touch Me > select a Family watch > School time.
  2. Set when you want to active school time mode in the morning (AM) and afternoon (PM) separately.
  3. Select which days of the week to active school time mode.
  4. Touch Save.
